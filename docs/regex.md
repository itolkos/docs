## 元字符
| 字符 | 说明                  |
||--|
| `.`  | 匹配任意单个字符      |
| `^`  | 匹配字符串的开始      |
| `$`  | 匹配字符串的结束      |
| `\d` | 匹配数字（等价于[0-9]）|
| `\w` | 匹配字母/数字/下划线  |
| `\s` | 匹配空白字符          |
| `\n` | 匹配换行符            |
| `\t` | 匹配制表符            |



## 量词
| 符号  | 说明                |
|-||
| `*`   | 0次或多次           |
| `+`   | 1次或多次           |
| `?`   | 0次或1次            |
| `{n}` | 精确匹配n次         |
| `{n,}`| 至少n次             |
| `{n,m}`| 匹配n到m次         |



## 分组与引用
```regex
(exp)         # 捕获分组
(?:exp)       # 非捕获分组
(?<name>exp)  # 命名分组（部分语言支持）
\1            # 反向引用第一个分组
```



## 字符类
```regex
[abc]     # 匹配a/b/c中任意字符
[^abc]    # 匹配非a/b/c的字符
[a-z]     # 匹配小写字母
[A-Z0-9]  # 匹配大写字母或数字
```



## 断言
| 类型        | 语法          | 说明                      |
|-|||
| 正向先行断言 | `(?=exp)`     | 匹配后面跟随exp的位置     |
| 负向先行断言 | `(?!exp)`     | 匹配后面不跟随exp的位置   |
| 正向后行断言 | `(?<=exp)`    | 匹配前面是exp的位置       |
| 负向后行断言 | `(?<!exp)`    | 匹配前面不是exp的位置     |



## 修饰符
| 修饰符 | 说明                  |
|--|--|
| `i`    | 不区分大小写          |
| `g`    | 全局匹配              |
| `m`    | 多行模式（影响^和$）  |
| `s`    | 使`.`匹配换行符       |



## 示例
1. 匹配邮箱：
```regex
^[\w-]+(\.[\w-]+)*@([\w-]+\.)+[a-zA-Z]{2,7}$
```

2. 替换日期格式（YYYY-MM-DD → DD/MM/YYYY）：
```regex
(\d{4})-(\d{2})-(\d{2})
替换为：\3/\2/\1
```



## 常用正则
- URL匹配：
```regex
^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$
```

- 中文匹配：
```regex
[\u4e00-\u9fa5]
```

- 手机号（中国大陆）：
```regex
^1[3-9]\d{9}$
```


## 注意事项
1. 使用`\`转义特殊字符（如`\.`匹配点号）
2. 不同编程语言的正则实现可能有细微差异
3. 使用在线工具（如regex101.com）测试正则表达式
